- Motivation topics selection (UI ready, persistence planned)

- Home screen (planned)

---

## 🛠️ Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- StateFlow
- Navigation Compose
- DataStore (Preferences) – integration in progress

---

## ⚙️ Requirements
- **Android Studio** Giraffe or newer
- **Gradle** 8.13+ (handled automatically)
- **Minimum SDK:** 24
- **Target SDK:** 34

---

## 🚀 Getting Started

#### 1. Clone the Repository

```bash
git clone https://github.com/minij4/Endo_App
```
#### 2. Open Android Studio
* Select File → Open and choose the project folder

#### 3. Sync Gradle
* Android Studio will automatically sync Gradle dependencies
* If prompted, install missing SDK components

#### 4. Run the app
* Select a device/emulator
* Click Run ▶ or press Shift + F10
