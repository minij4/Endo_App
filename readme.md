# 📱 Demo Endo App 

It is android application generated with kotlin + compose for upwork.com site for NewAge company for Tanav Patel.

---

## ✨ Functional

- Onboarding screen
- Languages choosing (back-end not working yet)
- Motivations topics choosing (back-end not working yet)
- Notification agreement

---

## 🛠️ Technologies

- Kotlin
- Jetpack Compose
- Material3
- DataStore (Preferences) (NOT EVERYTHING WORKS)

---

## ⚙️ Requirements

- Android Studio Hedgehog or newer
- minSdk 24+
- targetSdk 33+
